thymeleaf needs web link at the top. that is::-> xmlns:th="http://www.thymeleaf.org"

you can specify content language:

other necessary info like characterset, viewport, page title, site tab icon, css, meta data etc:



icons,images or svg should be inside webapp folder. you can create nested folder but refer correct path to html.

please include js if needed on body tag: to reduce page loading

normal html does not require extra action. if we use post mapping then, thymeleaf form require th:action="@{the name of the view that is called}".
for more details visit thymeleaf website
